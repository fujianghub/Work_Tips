# 目录处理.py

- 一个用于清理文本文件行尾冗余 "-数字" 的 Python 脚本，专为处理结构化文档设计

- ✨ 功能特性

  - **智能清理**  自动删除每行末尾的 `-数字` 格式冗余内容（如 `-167`）
  - **编码兼容**  支持 `utf-8`/`utf-8-sig`/`gbk`/`utf-16` 等多种编码格式
  - **无损处理**  保留原始缩进格式，仅修改目标内容
  - **安全备份**  生成新文件 `processed_原文件名.txt`，原始文件不受影响

🚀 快速使用

- 环境要求：Python 3.6+

- 使用步骤：
  1. 将脚本和待处理文件放在同一目录
  2. 运行命令：`python 目录处理.py`
